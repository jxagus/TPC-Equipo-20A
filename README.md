## – Gestión de BAR

Este proyecto es una aplicación web desarrollada en **C#**, con la función principal de permitir que un negocio gestione su stock, sus comandas y finanzas. Todo esta realizado con una arquitectura limpia en capas, donde todo se almacena en una base de datos **SQL** aplicando seguridad en todos los rincones de la app.

## 💻 Funciones

El administrador puede:  
- Visualizar, buscar y filtrar stock según el criterio que selecciones.
- Crear, modificar y eliminar usuarios
- Crear nuevas categorias y todo tipo de detalle que los platos necesiten.
- Crear mesas, utilizarlas, asignarle mozo, cerrarlas, etc.
- Acceder a una vista tipo GridView con todos los platos listados.
- Utilizar un filtro extra, donde puede hacer consultas mas avanazadas o especificas sobre las ganancias.
- Dashboard donde se puede ver que plato es el mas pedido, que mesa tiene mas rotacion, cuanto se recaudo por dia/mes/año, etc.
